# 🚀 PyGitUp v2.4.5 - Inline Help System

## 📖 NEW: Comprehensive Inline Help
- **Option H**: Get help for any feature
- Search by feature name or number
- `list` command shows all 45+ features
- Detailed descriptions for every operation

## 🔧 Bug Fixes
- Fixed file upload crash with better error handling
- Graceful JSON parsing fallbacks
- Improved SHA retrieval

## 📊 Previous Updates Included:

### v2.4.4 - Multi-Language SBOM
- Detects Python, Node.js, Rust, Go, Java, Ruby, PHP
- Auto-detects project name from package.json, Cargo.toml
- Warns about Python-only SBOM limitation
- Suggests alternative tools for multi-language projects

### v2.4.3 - Universal SBOM
- SBOM now works with ANY project (not just PyGitUp)
- Auto-detects project name/version
- Filters to project dependencies only

### v2.4.2 - Android/Termux Fix
- Replaced inquirer with questionary (10 files)
- All interactive prompts now work on Android
- Progress bars for long operations
- Time estimates & session caching
