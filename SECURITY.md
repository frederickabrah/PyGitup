# Security Policy

## Supported Versions

We are committed to providing security updates for the latest major version of PyGitUp. Please ensure you are using the most recent release to receive the latest security patches.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

The security of PyGitUp is a top priority. We appreciate your efforts to responsibly disclose your findings, and we will make every effort to acknowledge your contributions.

To report a security vulnerability, please send an email to **frederickabraham547@gmail.com** with a detailed description of the issue, the steps to reproduce it, and any relevant logs or screenshots.

Please **do not** report security vulnerabilities through public GitHub issues.

We will work with you to resolve the issue and will notify you when a patch is available. We will also credit you for your discovery unless you prefer to remain anonymous.

Thank you for helping to keep PyGitUp secure.
